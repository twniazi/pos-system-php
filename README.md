# pos-system-php
A role-based POS management system developed using core PHP and MySQL, featuring secure authentication, admin management, and database-driven operations for efficient business workflow handling.
# POS System Setup

1. Clone this repository:
   git clone https://github.com/twniazi/pos-system-php.git

2. Create a MySQL database named `pos_system`.

3. Import the SQL file:
   Open phpMyAdmin → Select database `pos_system` → Import → Choose `database/pos_system.sql`

4. Update DB credentials:
   config/dbcon.php → Enter your MySQL username & password

5. Run the project:
   Open XAMPP → Start Apache & MySQL → Access project via browser
